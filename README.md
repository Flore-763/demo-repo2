## Some useful commandes in github

git add .
git commit -m "expliquer ce qui a été fait" -m "ajouter une description"
git push -u origin main ou master selon le nom de la branche principale
git clone lien ssh pour cloner un repos distant
git init lien ssh pour joindre un repos local a un repo qui a ete cree sur le github
git checkout nom de la nouvelle branche que l'on veut creer
git status pour connaitre le statut de notre git , savoir s'il ya des commits ou des push a faire
